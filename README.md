## Setup

macos: Follow instructions here to set up chruby: https://jekyllrb.com/docs/installation/macos/

```sh
brew install chruby ruby-install xz
ruby-install ruby 3.3.3
```

then: 

```sh
bundle install
bundle exec jekyll serve
```
